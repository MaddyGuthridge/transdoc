# Contributing to Transdoc

I'd love to have your contributions!

## Development setup

This project uses [`uv`](https://docs.astral.sh/uv/) as a package manager.

Install dependencies

```sh
uv sync
```

Run linting

```sh
uv run ruff check
```

Run test suite

```sh
uv run pytest
```

Run type-checking

```sh
uv run mypy
```

Build package

```sh
uv build
```
